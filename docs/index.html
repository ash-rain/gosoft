Features

🔍

Multi-Format Support

Load ELF, PE, Mach-O, and .NET/Mono assemblies. Auto-detects format from magic bytes and CLR metadata.

🧠

AI Decompilation

Uses local Ollama models to produce clean, idiomatic code in Go, C, Rust, Python, TypeScript, Java, or C#.

⚡

Instant Offline Preview

Quick pseudo-code decompilation without AI. Browse functions and see translated pseudo-code instantly.

⌨️

Keyboard-Driven TUI

Navigate symbols, disassembly, hex, and source with Vim-style keys. Filter, search, and chat — all from the terminal.

🔧

Built-in Disassemblers

Pure Go disassemblers for x86/x64, ARM/ARM64, and ECMA-335 IL. No external dependencies like capstone.

💬

AI Chat

Ask questions about the binary — "what libraries does this use?" or "find all network calls" — right inside the TUI.

Quick Start

Install

git clone https://github.com/ash-rain/gosoft.git
cd gosoft
go build -o godecomp ./cmd/godecomp

Explore a binary

godecomp explore /usr/bin/ls
godecomp explore MyApp.exe
godecomp explore MyAssembly.dll

Decompile from CLI

# Single function
godecomp decompile /usr/bin/ls --func main --lang go

# Stream token-by-token
godecomp decompile /usr/bin/ls --func main --lang rust --stream

# All functions to a directory
godecomp decompile MyApp.exe --all --lang csharp --out ./src/

Other commands

godecomp symbols /usr/bin/ls               # List functions
godecomp strings /usr/bin/ls               # Extract strings
godecomp ask /usr/bin/ls "what does main do?"  # Ask AI
godecomp models                            # List Ollama models
